Hello, my name is Evaldo Borba and I am a Computer Engineer. I got a Bachelors and Masters in Computer Engineering by specializing in Machine Learning, and currently I have been employed making games and applications mainly targeted to VR/AR/MR devices.

During my free time, I create tools and gameplay systems to make the development of games and applications easier and faster, while making sure to write clean and maintainable code.

Also, my github page contain a good variety of projects of different types, like machine learning, data analysis, programming challenges and so on. So if you're interested in any of these topics, please check it out.
